# niow

This repository adds a minimal scaffold for the `niow` tool.

Quick start

- Install: pip install -e .
- Run: niow --name alice

Files added

- niow/ (package)
- pyproject.toml
- tests/
- Dockerfile
- scripts/run.sh

Purpose

This scaffold provides a working CLI entry point, a simple core module, a unit test, and a Dockerfile so you have a "physical copy" to inspect and run immediately.
